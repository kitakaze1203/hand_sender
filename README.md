# hand_receiverを含めた全体の概要
このプロジェクトは所属しているロボット研究会での部内の大会で使用したシステムです。\
大まかな通信の流れとして
1. スマートフォン（Android）にインストールしたhand_senderからUDPで送信
2. hand_receiverで受信。ベクトルを計算しロボットに載っている無線通信用マイコンへUDPで送信
3. 無線通信用マイコンから各マイコンへ有線通信
といったものになっています。

# このアプリの使い方
1. 送信先のノートPCと同じWi-Fiに繋がっていることを確認する。
2. 画面のInputfieldに送信先のIPv4アドレスを入力する。
3. 操作方法に準じてロボットを操作する

### 各種操作
- 前後左右の移動 → スマートフォンを前後左右に傾ける。
- 旋回 → 地面と水平にスマートフォンを回転させる。
- 各種モータ → 各モーター番号に対応したボタンを押し続ける。ONで正転、REで逆転する。
- エアー(空気の噴出と吸引によるピストン) → airと書いてあるボタンを押す。
- IPアドレスを再設定する → 左上の"IP設定に戻る"ボタンを押す。
- 傾きのオフセット → "offset"ボタンを押すと押した瞬間の傾きを零点とする。押している間は0を送信し続けるため擬似的な無操作状態にすることも可能。
